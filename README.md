## 概要
本リポジトリは、JavaScript（Vanilla JS）を用いて作成した「鬼滅の刃」キャラクター一覧表示アプリです。
外部 API を利用してキャラクター情報を取得し、画面上に一覧表示する動的な描画を実装しています。 
外部ライブラリに依存せず、DOM 操作・イベント制御・非同期処理を自前で実装しています。

## 使用技術
- JavaScript（Vanilla JS）
- HTML / CSS

## 機能一覧
- 外部 API から「鬼滅の刃」キャラクターのデータを取得して表示
- 取得したキャラクター名と画像の一覧表示
- DOM を使った動的描画
- データ取得時にローディング画面を表示
